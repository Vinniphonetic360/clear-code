## 🧹 Common problems

If the app does not open:

- Make sure the file finished downloading
- Try right-clicking the app and choosing **Run as administrator**
- Check that Windows did not block the file
- Move the file to a simple folder like Downloads or Desktop
- Restart your PC and try again

If your logs do not show up:

- Check that Claude Code created the log files
- Make sure the app can read the log folder
- Refresh the dashboard
- Open the right log source

If the screen looks blank:

- Resize the window
- Restart the app
- Open a newer log file
- Check whether the log data is empty
